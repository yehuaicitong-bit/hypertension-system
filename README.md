# 基于大数据分析的高血压患病风险评估与预测系统

## 项目简介
本项目基于2000–2024年全国高血压流行病学监测数据，构建多维度风险评估模型与趋势预测系统，为公共卫生决策、慢病防控、政策干预提供数据支撑与可视化工具。

## 功能模块
- **首页概况** — 实时风险指标展示（总体患病率、老年占比、性别/城乡患病率、综合风险指数）
- **数据图表** — 性别/城乡/年份趋势对比
- **数据洞察** — 关键统计与结论（含知晓率/治疗率/控制率）
- **中国地图** — 全国患病率空间分布
- **风险雷达图** — 多因素风险结构分析
- **政策模拟** — 干预效果量化评估
- **趋势预测** — 基于线性回归的未来预测（2026年）
- **项目介绍** — 数据来源与技术说明

## 核心技术
- **Streamlit** 可视化框架
- **Pandas/Numpy** 数据处理
- **Matplotlib/Pyecharts** 绘图
- **Scikit-learn** 机器学习预测
- **Z-score 标准化风险合成模型**（权重：总体患病率0.4，老龄化0.25，性别差异0.2，城乡差异0.15）
- **政策干预模拟算法**

## 数据来源
1. 2025年《Cell》子刊：全国18岁及以上成人高血压患病率31.6%（阜外医院王增武团队）
2. 2018年《中华地方病学杂志》Meta分析：西藏、海南及区域合并患病率
3. **李镒冲等《中华预防医学杂志》2012**：2010年全国成年人高血压患病率33.5%（样本量98548）
4. **BMJ 2023**：知晓率、治疗率、控制率（2004-2018年趋势，2018年三率分别为38.3%、34.6%、12.0%）
5. **李永泉等《中国公共卫生》2014**：2008年城乡患病率（城市21.24%、农村19.33%）
6. 中国慢性病及危险因素监测、中国高血压调查、中国居民健康状况调查
7. 城乡差异定义为城市患病率减农村患病率，反映城乡风险对比。
