# 基于大数据分析的高血压患病风险评估与预测系统

## 项目简介
本项目基于2000–2024年全国高血压流行病学监测数据，构建多维度风险评估模型与趋势预测系统，
为公共卫生决策、慢病防控、政策干预提供数据支撑与可视化工具。

## 功能模块
1. 首页概况 — 实时风险指标展示
2. 数据图表 — 性别/城乡/年份趋势对比
3. 数据洞察 — 关键统计与结论
4. 中国地图 — 全国患病率空间分布
5. 风险雷达图 — 多因素风险结构分析
6. 政策模拟 — 干预效果量化评估
7. 趋势预测 — 基于线性回归的未来预测
8. 项目介绍 — 数据来源与技术说明

## 核心技术
- Streamlit 可视化框架
- Pandas/Numpy 数据处理
- Matplotlib/Pyecharts 绘图
- Scikit-learn 机器学习预测
- Z-score 标准化风险合成模型
- 政策干预模拟算法

## 数据来源
- 2025《Cell》子刊全国高血压调查
- 2018《中华地方病学杂志》Meta分析
- 国家公共卫生监测数据
-中国慢性病及危险因素检测
-中国高血压调查
-中国居民健康状况调查
